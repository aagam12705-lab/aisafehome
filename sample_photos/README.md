# Sample Photos

Use only staged, non-patient photos for testing AI SafeHome.

Do not include:

- Faces
- Names
- Addresses
- Mail
- Bills
- Medication bottles
- Medical documents
- Real patient photos

Good staged test photo ideas:

- Safe living room
- Loose rug
- Cord across walkway
- Floor clutter
- Dim hallway
- Bathroom mat
- Stairs
- Narrow pathway
